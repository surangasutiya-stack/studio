# **App Name**: Connectify

## Core Features:

- User Authentication: Register and log in using phone number and password. Note: Registration will not involve verification codes for MVP purposes.
- One-to-One Chat: Send and receive text messages with individual contacts.
- Group Chat: Create and participate in group chats with multiple users.
- Real-time Messaging: Instant message delivery using websockets for a responsive chat experience.
- Contact List: Display a list of available contacts from your database to start chatting with.
- Search contacts: Ability to find your friends using their names
- AI Message Summarization: Summarize conversation threads using LLM models to quickly catch up on missed messages.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) to convey trust and reliability.
- Background color: Very light indigo (#E8EAF6).
- Accent color: Vivid magenta (#E91E63) for notifications and interactive elements.
- Body and headline font: 'Inter' sans-serif for a modern, neutral, and readable interface.
- Simple, outlined icons for easy recognition and a clean interface.
- Clean and intuitive layout with clear separation between chat threads and individual messages.
- Subtle animations for message sending and receiving to provide a fluid user experience.
